Grade Calculator

Developed a Grade Calculator, a desktop application built using Java with javax.swing and java.awt for an intuitive graphical user interface. The tool allows users to calculate total marks and grades for subjects efficiently. Interactive dialogue boxes were implemented to enhance the user experience by providing real-time alerts for invalid or missing inputs, ensuring smooth and error-free operation. This project highlights a strong understanding of GUI design principles and user-centered application development in Java.
